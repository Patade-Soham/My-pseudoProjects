# 📄 PRD — Midnight Bites: Luxury Fine Dining Website

## 1. Overview
**Product Name:** Midnight Bites  
**Type:** Multi-page Responsive Website  
**Stack:** React (Vite) + Tailwind CSS + Framer Motion  
**Event:** FrontFrame — 1-hour Landing Page Challenge  
**Theme:** Luxury Fine Dining with Futuristic Dark Aesthetics  

---

## 2. Objective
Build a visually stunning, fully responsive multi-page website for a luxury fine dining restaurant called **Midnight Bites**. The site must feel premium, immersive, and futuristic — using dark backgrounds, gold/amber neon accents, cinematic typography, and smooth animated transitions.

---

## 3. Target Audience
- High-income food enthusiasts
- Couples seeking premium dining experiences
- Corporate clients for exclusive events
- Food bloggers & critics

---

## 4. Pages

| Page | Route | Purpose |
|------|-------|---------|
| Home | `/` | Hero, intro, mood-setter |
| Menu | `/menu` | Full course menu display |
| Reservations | `/reservations` | Table booking form |
| About | `/about` | Story, chef, philosophy |
| Contact | `/contact` | Location, contact form |

---

## 5. Core Sections (per FrontFrame requirements)

| Component | Page |
|-----------|------|
| Header + Navigation | All pages |
| Hero Section | Home |
| About Section | About |
| Menu / Services Section | Menu |
| Team Grid (Chefs) | About |
| Testimonials | Home |
| Contact Form | Contact |
| Footer | All pages |
| Theme Toggle (Dark/Gold mode) | Header |
| Mobile Drawer Menu | Header (mobile) |

---

## 6. Key Features
- **Cinematic Hero** with video/parallax background and animated tagline
- **Animated Menu Cards** with hover glow effects
- **Reservation Form** with smooth validation feedback
- **Chef Profiles** in an asymmetric grid
- **Testimonials Carousel** with fade transitions
- **Dark/Gold theme toggle**
- **Mobile-first** responsive layout with hamburger drawer
- **Page transitions** using Framer Motion

---

## 7. Success Criteria (Evaluation Alignment)

| Criterion | How We Address It |

---

## 8. Constraints
- 1-hour build time
- No Next.js (use React + Vite)
- Must work on Chrome/Safari
- No backend — forms are UI-only
